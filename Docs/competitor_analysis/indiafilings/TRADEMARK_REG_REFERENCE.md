# Trademark Registration Analysis

## 1. Overview
Protecting brand identity (Logo, Wordmark, Slogan) under 45 different classes.

## 2. Input Requirements
### Applicant Details
- **Type:** Individual / Startup / Small Enterprise / Other
- **Name & Address**
- **Nationality**

### Mark Details
- **Type of Mark:** Word, Device (Logo), Sound, 3D
- **Brand Name / Slogan**
- **Class(es):** The specific category (Class 1-45) the goods/services fall under.
- **User Date:** "Proposed to be used" OR "Used since [Date]" (requires affidavit).

## 3. Document Checklist
### For Individual/Sole Proprietor
- **ID Proof:** PAN Card / Aadhaar / Passport
- **Authorization:** Form TM-48 (Power of Attorney) - Signed
- **Logo Image:** JPEG format

### For SME/Startup (50% Govt Fee Discount)
- **Udyam Registration Certificate** (MSME) OR **Startup India Certificate**
- **Incorporation Certificate**
- **Form TM-48** (Signed by Director)

## 4. User Flow
1.  **Trademark Search:** User checks if the name is available (Public Search).
2.  **Class Selection:** User identifies the correct class.
3.  **Form Drafting:** Application (Form TM-A) is drafted.
4.  **Filing:** Application filed with IP India.
5.  **Receipt:** "TM" number generated immediately. User can start using "TM".
6.  **Examination:** Registrar examines the mark (takes 1-3 months).
7.  **Journal Publication:** If accepted, published in Trademark Journal.
8.  **Registration:** If no opposition for 4 months, "R" certificate is issued.
