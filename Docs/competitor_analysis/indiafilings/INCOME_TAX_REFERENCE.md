# Income Tax & TDS Services Analysis

## 1. Overview
Filing of direct tax returns for individuals and businesses, along with Tax Deducted at Source (TDS) compliance.

## 2. Income Tax Return (ITR) Filing
### Categories
1.  **Salaried (ITR-1/2):** For individuals with salary, house property, and other sources income.
2.  **Business/Profession (ITR-3/4):** For proprietors and professionals (Presumptive Taxation).
3.  **Companies/LLPs (ITR-5/6):** Mandatory audit cases and corporate filings.
4.  **Capital Gains:** For income from sale of stocks, mutual funds, or property.

### Input Requirements
- **Form 16:** From employer (for salaried).
- **Form 26AS / AIS:** Annual Information Statement (downloaded from portal).
- **Bank Statements:** For interest and other income.
- **Investment Proofs:** 80C (LIC, PPF), 80D (Medical Insurance), HRA rent receipts.

### User Flow
1.  **Document Upload:** User uploads Form 16 and bank statements.
2.  **Draft Calculation:** Tax expert prepares computation.
3.  **Review:** User approves the tax liability/refund amount.
4.  **Filing:** Expert files return on e-filing portal.
5.  **Verification:** User e-verifies using Aadhaar OTP.

## 3. TDS Return Filing
### Input Requirements
- **TAN Number:** Tax Deduction and Collection Account Number.
- **Deduction Details:** Excel sheet with PAN of deductees, amount paid, and TDS deducted.
- **Challan Details:** BSR Code, Challan Serial Number, Date of Deposit.

### Forms
- **Form 24Q:** Salaries (Quarterly).
- **Form 26Q:** Other than Salaries (Contractors, Rent, Professional Fees) (Quarterly).
- **Form 27Q:** Payments to Non-Residents.

### Deliverables
- **Form 16/16A:** Certificates generated from TRACES portal after filing.
- **Filing Receipt.**

## 4. Notice Management
- **Section 143(1):** Intimation handling.
- **Scrutiny Assessment:** Replying to tax officer queries.
- **Rectification:** Correction of errors in filed returns.
